O objetivo deste projeto é elaborar uma aplicação móvel com tecnologia WEB;

  • HTML5
  • CSS3
  • JavaScript (Ecma Script 6)
  • Bootstrap 4
  • Apache Cordova
  • AJAX (Consumir API)
  • Git & GitHub

Bem como consumir serviço (API) disponíveis na Internet

Vamos criar uma aplicação WEB para dispositivo móvel e empacota-lo com Apache Cordova para smartphone (opcional).

O projeto pode ser feito com o editor de texto (Brackets ou MS VS Code), Live Preview, e com o Browser (DevTools - Ctrl + Shift + i).

Critérios de avaliação:

  • Design
  • Consumir API REST/JSON com JavaScript (AJAX)
  • Criar elementos HTML com base nos dados da API e exibir na página HTML
  • Coragem (Ref.: Extreme Programming Explained: Embrace Change, 2004, Kent Beck, Cynthia Andres)
